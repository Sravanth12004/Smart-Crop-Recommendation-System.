# 🌱 Smart Crop Recommendation System
A Machine Learning-based AgTech application designed to recommend the most optimal crop for cultivation based on soil metrics and environmental parameters.

## 📊 Dataset Features Used
- **N, P, K:** Nitrogen, Phosphorus, and Potassium ratios in the soil.
- **pH:** Acidity or alkalinity of the soil.
- **Climate Data:** Temperature (°C), Humidity (%), and Rainfall (mm).

## 🛠️ Tech Stack & Algorithms
- **Language:** Python
- **Environment:** Google Colab / Jupyter Notebooks
- **ML Algorithm:** Random Forest Classifier (Scikit-Learn)
- **UI Framework:** IPyWidgets for interactive browser sliders

## 🚀 How to Run
1. Open the `.ipynb` notebook file in Google Colab.
2. Upload the `Crop_recommendation.csv` dataset.
3. Run all cells to launch the interactive UI dashboard.# Smart-Crop-Recommendation-System.
